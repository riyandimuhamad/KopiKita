# KopiKita - Sistem Manajemen Menu Cafe

Aplikasi berbasis web untuk mengelola daftar menu kopi (CRUD) yang dikembangkan sebagai tugas besar mata kuliah Praktikum Pemrograman Web.

## Fitur Utama
Aplikasi ini sudah mendukung siklus **CRUD** lengkap:
* **Create**: Menambah menu baru (Nama, Harga, Kategori, Status).
* **Read**: Menampilkan katalog kopi dengan desain card yang estetik.
* **Update**: Mengubah data menu jika terjadi kesalahan input atau perubahan harga.
* **Delete**: Menghapus menu dari database dengan konfirmasi.

---

## Teknologi yang Digunakan
* **Framework**: Laravel 11
* **Styling**: Tailwind CSS
* **Database**: MySQL
* **Server**: Laragon

---

## Cara Menjalankan Project
1. Clone repository ini.
2. Jalankan `composer install` dan `npm install`.
3. Buat database di MySQL dengan nama `kopi_kita`.
4. Jalankan `php artisan migrate`.
5. Jalankan `php artisan serve` atau buka lewat Laragon.
